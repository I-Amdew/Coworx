# Computer Use Safety Checklist

## Before Starting
- Confirm the exact app and task.
- Confirm action level and approval.
- Close unrelated sensitive apps when practical.
- Do not use accounts, credentials, payments, security settings, or destructive actions.
- Define stop conditions before interacting.

## Safe First App
Calculator is approved for harmless local readiness tests.

## Stop Immediately If
- Permission prompt appears.
- Wrong app is active.
- The UI asks for login, credentials, 2FA, payment, account security, or private data.
- The next action would submit, publish, delete, purchase, invite, merge, deploy, or change settings.

## Evidence To Record
- App name.
- UI state observed.
- Exact actions taken.
- Result observed.
- Whether any stop condition appeared.
