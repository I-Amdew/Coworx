# Meeting To Deliverable Playbook

## Purpose
Turn meeting recordings, transcripts, notes, or chat exports into useful work products.

## Default Action Level
2 for local files. Level 3 for external drafts. Level 4 for sending follow-ups or scheduling meetings when delegated or explicitly approved.

## Routing
1. Audio/video: use Transcribe.
2. Summary document: use Documents.
3. Slide deck: use Presentations.
4. Action tracker: use Spreadsheets.
5. Follow-up message: draft locally; send or schedule only with delegated authority or explicit approval.

## Review Checklist
- Names and dates are accurate.
- Decisions and action items are separated.
- Sensitive content is kept in private outputs if needed.
- No message, invite, schedule, or external publish occurs outside delegated authority or explicit approval.

## Stop Conditions
- Missing permission to use the source file.
- Sensitive personal, medical, legal, financial, or employment content.
- Request to send, invite, schedule, or publish outside delegated authority or explicit approval.
- Level 5/protected action requested.
