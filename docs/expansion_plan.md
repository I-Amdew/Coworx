# Expansion Plan

## Phase 1: Filesystem
Build folders, docs, templates, skills, and policies.

## Phase 2: Internal Task Runner
Run one local task from `queue/todo/`, write logs, review output, and update safe memory.

## Phase 3: Playwright Demo
Use a public demo site with no login and no private data. Save action request, result, browser map, selectors, run log, final report, and lessons.

## Phase 4: Computer Use Readiness
Create checklists and a harmless local-app test plan. Do not operate real accounts.

## Phase 5: App Mapping
Map one real app or site read-only after explicit approval and manual login by the user.

## Phase 6: Real Workflow
Perform a low-risk draft-only task such as drafting an issue or summarizing dashboard data.

## Phase 7: Regression
Add repeatable smoke tests, workflow tests, and review checklists.
