# Final Action Reporting

Every Coworx task that uses browser, Computer Use, accounts, files, plugins, or external systems must end with a clear action ledger.

## Required Sections
- actions taken;
- tools/plugins used;
- accounts/apps/sites touched;
- files read;
- files written;
- private outputs created;
- screenshots/traces captured;
- external transmissions;
- approvals used;
- stop conditions encountered;
- memory updates written;
- sanitized lessons proposed;
- residual risk.

## External Action Disclosure
If Coworx sent, submitted, published, uploaded, scheduled, invited, moved, deleted, merged, deployed, or changed settings, the final report must list:
- exact action;
- target;
- data changed or transmitted;
- approval record;
- timestamp;
- result;
- undo notes if known.

## No-Action Disclosure
If Coworx stopped at a draft or final button, say that plainly.
