# Coworx Config Templates

This directory holds shippable templates for approved sites, autonomy grants, browser lanes, and action levels.

Do not store real private account details or secrets here. User-specific grants and account maps belong in ignored private paths unless sanitized.

Use these templates to define:

- which site/app/account label Coworx may use;
- which actions are allowed without another question;
- which actions must be staged;
- which actions are always blocked;
- whether notifications, invitations, sends, submits, uploads, or external writes are delegated;
- required evidence and expiration.
